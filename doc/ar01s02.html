2. Installation, désinstallation

2.1. Sous Linux

Georges Khaznadar, qui se charge de l’empaquetage de Collatinus sous Linux, a construit pour Debian, Ubuntu et autres distributions dérivées, des paquets *.deb. Il a également prévu, pour RedHat, Mandriva et dérivées, des paquets *.rpm. Ces paquets s’installent et se désinstallent selon les usages de leur distribution.

2.2. Sous Macintosh

Une version de Collatinus existe pour Macintosh, sensiblement différente de celle-ci. Voir à l’adresse http://www.infx.info/quidnovi/article.php3?id_article=792

2.3. Sous Windows

L’installeur est disponible sur le site de l’académie de Poitiers

http://www.ac-poitiers.fr/lettres/lang_anc/Collatinus.htm

2.4. Pour désinstaller

Il faut localiser le répertoire d’installation, en général

C:\Program Files\collatinus\

et lancer le programme uninst.exe.

2.5. À Partir des sources

Vous pouvez récupérer les dernières sources à l’adresse ci-dessous

https://github.com/Carreau/Collatinus

Ou si vous êtes déja utilisateur de git:

git clone https://Carreau@github.com/Carreau/Collatinus.git
